# Cleopatra Delights — Business Resources

This repository contains business planning materials and the website scaffold for **Cleopatra Delights**, a Jacksonville, FL food-trailer and online-resale operation.

## Contents

| File / Folder | Description |
|---|---|
| [`action-plan.md`](./action-plan.md) | Complete 6-month business action plan: eBay launch, wholesale accounts, Amazon FBA |
| `cleopatra-delights/` | Static website scaffold (HTML / CSS / JS) — see that folder's README for run instructions |

## Quick start — action plan

Open [`action-plan.md`](./action-plan.md) for step-by-step tasks organised by phase:

- **Phase 1 (Week 1):** Florida resale certificate · eBay seller account · first 10 listings
- **Phase 2 (Weeks 2–4):** Wholesale distributor accounts · first wholesale order
- **Phase 3 (Months 2–3):** Amazon ungating submission · FBA launch
- **Phase 4 (Months 4–6):** Scale, systematize, evaluate next steps

## Quick start — website

```bash
cd cleopatra-delights
python3 -m http.server 8000
# open http://localhost:8000
```
