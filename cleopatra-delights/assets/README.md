## Photography Tone Guidelines

- **Background surfaces:** dark slate, matte black marble, deep linen — match the `#1a1a1a` site background
- **Lighting:** controlled, directional; avoid harsh shadows or bright white reflections
- **Colour palette:** muted warm tones; no heavy orange food-blog warmth
- **HDR/Sharpening:** minimal — editorial, not commercial catalogue style
- **Format:** Progressive JPEG, quality 82–88, sRGB colour profile

## How to Replace

1. Export each photo at the recommended dimensions.
2. Name the file to match the placeholder filename shown above.
3. Drop the new file into this `assets/` folder (overwriting the placeholder).
4. Re-verify `alt` text in `index.html` accurately describes each new image.

*Note: If you change file names, update the corresponding `src` attributes in `index.html`.*
